# 오픈소스소프트웨어설계및실습 <br/>프로젝트 제안서

## 프로젝트 소개

## 개발 환경
- JavaScript
- HTML
- CSS
  
## 주요 기능
- 광운대학교 주변 식당을 자료구조화해 저장
- 식당을 다양한 기준에 따라 정렬하여 열람
- 룰렛 등 추첨 기능으로 식사 메뉴 추천
